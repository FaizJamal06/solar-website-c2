# Kotieswara Associates Solar Landing Page

[![Vercel Deployment](https://img.shields.io/badge/Vercel-Deployed-black?logo=vercel&logoColor=white)](https://solar-website-c2.vercel.app/)

A premium, modern single-page showcase landing page for **Kotieswara Associates** — Your Trusted Solar Partner.

Live Website URL: **[https://solar-website-c2.vercel.app/](https://solar-website-c2.vercel.app/)**

## Context of the Site
This website has been rebranded from its previous template code name "HelioNest" to **Kotieswara Associates**, an Indian solar energy solutions company. The landing page is designed to maximize conversion rates for solar installations, combining premium navy-and-white palettes, sleek micro-animations, and a highly responsive structural outline.

### Major Features
1. **Interactive Hero Section**: Features custom headlines and an **inline Quote Enquiry Form** allowing visitors to easily input details (Name, Phone, Pincode, Product, System specifications) to request a call.
2. **Dedicated Products Showcase**: High-efficiency specifications details for:
   - Solar Rooftop Systems
   - Solar Street Lights
   - Solar Water Pumps
3. **Services Section**: Breakdown of corporate service pipelines:
   - Supply & Distribution (MNRE & ISI certified parts)
   - Certified Installation
   - Lifecycle Maintenance (AMC contracts)
4. **Interactive Savings Calculator**: Estimates system size capacity, annual savings, and lifetime return on investment (payback period of 3-4 years) dynamically based on utility bill ranges.
5. **Real-world Testimonial Boards & FAQs**: Clean customer reviews and answers to standard questions about warranty, government subsidies, and monsoon generation capacity.
6. **Mobile Sticky Bottom Form**: Provides a persistent, call-to-action panel for mobile clients to easily apply for solar surveys.
7. **Premium Design Language**: Formatted with Google Fonts (`Syne` and `Inter`), CSS Grid layouts, and custom AI-generated process visuals.

## Project Structure
- `index.html` — Containing the entire markup, custom styles, interactive components, responsive stylesheets, and validation/calculator scripts.
- `./process_*.png` — High-quality custom AI concept artwork for the "How It Works" phase visual cards.
